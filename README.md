# MMSMCUDA

## REQUIREMENTS
The code is executable on the NVIDA graphics card and tested on GeForce RTX 2060. <br>

## COMMANDS TO RUN
```
nvcc main.cu -o a
sbatch sb.sbatch
cat o.out
```

## Output
e.err contains the error message. <br>
o.out contains all numbers of elements of result matrix.<br>
